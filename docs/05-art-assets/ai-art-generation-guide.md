## Generation Settings (if applicable)

- **Style**: Vector / Cartoon / 2D Game Asset
- **Aspect Ratio**: 1:1 (square)
- **Quality**: High
- **Details**: Medium (not too realistic, not too simple)
- **Variations**: Generate 3-4 per character, pick the best

---

## Post-Generation Checklist

After generating each character, verify:

- [ ] **Transparent background** (no white box)
- [ ] **Facing right** (same direction as cart movement)
- [ ] **Similar size** (head-to-body ratio matches others)
- [ ] **Consistent line weight** (outlines similar thickness)
- [ ] **Vibrant colors** (not washed out or too dark)
- [ ] **Happy expression** (friendly, approachable)
- [ ] **Simple details** (not overly complex)
- [ ] **512x512px minimum** resolution

## Image Editing After Generation

### If characters need adjustments:

**Tools:**
- **Remove.bg** - Remove backgrounds if not transparent
- **Photopea** (free) - Edit colors, resize, adjust
- **Inkscape** (free) - Vectorize if needed
- **Figma** (free) - Batch resize and export

**Common Adjustments:**
- Crop to same dimensions (512x512)
- Adjust brightness/saturation to match
- Add or thicken outlines
- Flip horizontally if facing wrong direction

## Ludo.ai Specific Tips

### Prompt Structure for Best Results:
1. **Subject** (cute [animal] character)
2. **Style** (flat vector art, cartoonish)
3. **Technical** (2D game asset, simple shapes)
4. **Visual** (bright colors, thick outlines)
5. **Composition** (facing right, transparent background)
6. **Mood** (family-friendly, happy expression)

### If Results Aren't Consistent:
- Use **"Remix"** feature to generate variations
- Try **"Style Transfer"** from your best character
- Adjust **prompt weight** for key terms
- Generate in **batches** (all 13 at once with same seed)
