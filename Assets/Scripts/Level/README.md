# Level Scripts

Scripts related to level mechanics, boundaries, and level-specific functionality.

## Scripts

- **FinishLine.cs** - Detects level completion (future)
- **ProceduralLevelGenerator.cs** - Generates dynamic levels (Month 2)
- **ChunkSpawner.cs** - Spawns pattern chunks (Month 2)
