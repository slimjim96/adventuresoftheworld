# ASSET VARIATION TIPS

## Creating Color Variants

To adapt cross-theme assets, modify color descriptions:

**Boulder for Mountain:**
```
Replace: "weathered gray stone with subtle moss patches"
With: "blue-gray stone with snow patches, alpine weathering"
```

**Boulder for Desert:**
```
Replace: "weathered gray stone with subtle moss patches"
With: "warm orange-red sandstone with wind-carved surface"
```

**Cliff for Ocean:**
```
Add: "tide-worn surface, subtle blue water reflections in crevices"
```

## Batch Generation Strategy

1. **Generate cross-theme assets first** - Boulders, cliffs, clouds, rocks
2. **Test one asset per theme** - Verify color adjustments work
3. **Batch by layer** - Generate all Far layers, then Mid, then Near
4. **Maintain consistent exclusions** - Always use the same `--no` parameters
